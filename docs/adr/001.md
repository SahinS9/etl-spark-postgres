Source PostgreSQL version: 17.10
Target PostgreSQL version: 18.4

Source platform: x86_64-pc-linux-gnu
Target platform: aarch64-unknown-linux-gnu

Migration method: PostgreSQL logical replication

Decision:
The target PostgreSQL 18 project will be retained. The migration will
simultaneously move the database from the deprecated Neon Azure region
and upgrade PostgreSQL from major version 17 to 18.

Additional controls:
- schema compatibility validation
- extension compatibility validation
- application and JDBC testing
- sequence synchronization
- source-to-target data reconciliation
- controlled cutover and rollback plan
